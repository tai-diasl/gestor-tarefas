
# Gestor de Tarefas

Este projeto é uma aplicação web de gestão de tarefas, desenvolvida com HTML, CSS e TypeScript. Permite adicionar, remover, editar e concluir tarefas, bem como organizar e realizar buscas por tarefas em tempo real.


## Funcionalidades

- Adicionar novas tarefas com título e categoria (Trabalho, Pessoal, Estudo).

- Editar o título de tarefas existentes.

- Marcar tarefas como concluídas ou desmarcadas.

- Remover tarefas da lista com confirmação.

- Ordenar tarefas alfabeticamente pelo título.

- Limpar todas as tarefas concluídas.

- Pesquisar tarefas em tempo real pelo título.

- Contador de tarefas pendentes com alertas quando o número aumenta.

- Indicação da categoria com cores diferentes para fácil visualização.

- Exibição da data e hora de conclusão das tarefas concluídas.

- **CSS3** - Estilo
- Manipulação do **DOM** e eventos com TypeScript puro.


## Como usar

1. Clone o repositório (no bash)

`git clone 
https://github.com/tai-diasl/gestor-tarefas.git`

2. Entre na pasta do projeto

`cd gestor-tarefas
`

3. Compile o TypeScript para JavaScript

`tsc
`

4. Abra o `index.html` em um navegador (Google Chrome, Firefox, etc).


### Uso
- Digite o título da tarefa e selecione a categoria, depois clique em Adicionar.

- Clique em Concluir/Desmarcar para alterar o estado da tarefa.

- Clique em Editar para alterar o título de uma tarefa.

- Clique em Remover para excluir uma tarefa da lista.

- Use o campo de pesquisa para filtrar tarefas pelo título.

- Clique em Ordenar para organizar as tarefas alfabeticamente.

- Clique em Limpar Concluídas para remover todas as tarefas concluídas.

- As Categorias das tarefas são exibidas com cores diferentes para melhor identificação.

- Tarefas concluídas mostram a data e hora da conclusão.
